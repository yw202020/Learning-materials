如果批文件已经成功的在你的计算机上安装了数据库
(表面的现象就是刚才你看到Dos窗口中的不停的翻屏)
你可以不必阅读下面的文件,直接将该窗口关闭，
完成数据库的安装工作。如果有问题，请往下读：

--------------------------------------------------------------------------
使用前提与常见问题

1、你的计算机已经成功的安装了SQL Server 2000。

2、你的SQL Server服务器已经启动。(在系统任务栏上存在SQL的绿色图标)

3、SQL Server系统允许Windows身份验证。

4、双击任务栏上的SQL图标，在服务器文本框中显示的名字仅仅是你的计算机的名字，
例如ZhangSan；而不是“计算机名\服务名”这样的形式，如“aiguozhe\zySQL”。

如果你的计算机满足上面的条件，恭喜你！本程序会自动的在你的计算机上
安装好CreditCard,OnlineBanking,Recruitment，ToyUniverse，VideoKiosk五个数据库。
--------------------------------------------------------------------------

--------------------------------------------------------------------------
如果是你的计算机不满足上述条件，请手工的安装数据库了，方法是：

1、启动你的查询分析器，并且连接到你的数据库服务器。

2、在查询分析器中打开并执行CreditCard.sql

3、在查询分析器中打开并执行OnlineBanking.sql

4、在查询分析器中打开并执行Recruitment.sql

5、在查询分析器中打开并执行ToyUniverse.sql

6、在查询分析器中打开并执行VideoKiosk.sql
--------------------------------------------------------------------------

通过上面的方法，你应该能够在自己的计算机上部署五个数据库了。
